# Changelog

All notable changes to **OW Smurfer** are documented in this file.

## [2026-05-01]

### Added

- Added repository documentation with layout, runtime data, workflow, and maintenance notes.
- Added this changelog as the project history entry point.

### Current Behavior

- Provides a compact Windows tray UI for saved Overwatch account login.
- Supports a quick account selector, configurable hotkey, Enter or Tab field mode, and optional Windows startup.
- Stores obfuscated local account data under `%LOCALAPPDATA%\OW_Smurfer`.
